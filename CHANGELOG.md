# Changelog

## 2.0.0

- Rebuilt the library around modular generator and core packages.
- Added 26 supported CAPTCHA generators.
- Added shared configuration, accessibility metadata and render hooks.
- Added standalone simple and Tkinter examples for every CAPTCHA.
- Added image, logic, puzzle, maze, clock and 3D challenges.
- Improved sequence, rotation, pattern and selection challenge correctness.
- Added Python 3.11 through 3.14 support.
- Removed the legacy resource pack and Telegram-focused project files.
